# 프론트엔드 UI 개선 및 로그인 구조 구축 회고록

**일자**: 2026-05-14  
**담당**: agami MLOps

---

## 홈화면 반응형 수정

노트북 환경에서 버튼이 잘리는 문제, UI 겹침 현상, 스크롤 오류가 발생하고 있었다.

**해결**
- 레이아웃 구조 수정
- overflow 처리 개선
- 반응형 UI 조정

---

## 물고기 인터랙션 개선

AGAMI 메인 화면의 물고기 인터랙션에서 커서 위치와 실제 반응 위치가 어긋나는 문제, 물고기 생성 위치가 부자연스러운 문제가 있었다.

**해결**
- 화면 전체에서 자연스럽게 움직이도록 개선
- 실제 마우스 위치와 반응 좌표 일치하도록 수정

---

## 로그인 구조 구축

JWT 기반 로그인 구조를 도입했다.

- FastAPI JWT 토큰 발급 기능 추가
- 카카오 로그인 성공 시 accessToken 반환
- 프론트 localStorage 기반 로그인 상태 관리
