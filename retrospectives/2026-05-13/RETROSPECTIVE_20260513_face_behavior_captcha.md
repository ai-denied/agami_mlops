## 방향 전환

기존에는 단순 얼굴 인식 기반 CAPTCHA를 고려했지만, 점점 더 중요한 것은 얼굴 자체보다 **사람다운 행동 패턴**이라는 방향으로 생각이 바뀌기 시작했다.

- 윙크
- 고개 움직임
- 반응 타이밍
- 자연스러운 landmark 변화

같은 요소들이 단순 정적 이미지보다 훨씬 중요하다고 판단했다.

---

## 행동 데이터 구조 설계

학습 데이터를 다음 3가지 유형으로 분리해서 설계하기 시작했다.

1. **실제 사용자 데이터** — 정상 사용자의 자연스러운 행동 데이터
2. **공격 / 봇 데이터** — 영상 재생, 가짜 얼굴, 반복 패턴 등 spoof 공격 데이터
3. **실패 / 예외 데이터** — 어두운 조명, 안경, 마스크, 저화질 카메라, 얼굴 일부 가림 등 실제 환경 문제 상황

---

## SiW Dataset 조사 및 선정

실시간 얼굴 행동 CAPTCHA 학습용 후보 데이터셋으로 **SiW Dataset**을 가장 적합한 후보로 선정했다.

선정 이유:
- 영상 기반 데이터셋
- 다양한 조명과 포즈 포함
- replay attack 데이터 존재
- landmark 기반 행동 feature 추출에 적합

---

## 고민한 문제

공개 데이터셋은 대부분 연구 목적 라이선스, 재배포 제한, 상업적 사용 제한 등의 문제가 존재했다. SiW 데이터셋 자체도 접근 승인이 필요할 가능성이 있었다.

## 미해결 / 다음 단계

장기적으로는 실제 CAPTCHA 서비스 환경에서 직접 행동 데이터를 수집하는 방향이 필요하다고 판단했다. 공개 데이터셋은 초기 모델 구축용으로 활용하고, 최종적으로는 실제 사용자 행동 데이터 기반으로 학습 구조를 전환해야 한다.
